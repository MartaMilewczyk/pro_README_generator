  ## Installation

In order to use this app please execute following steps:

- Install <a href="https://nodejs.dev/en/download/">node.js</a> 

- Run commands:

      $ npm init -y
      $ npm install

- Install npm's Inquirer and fs module

      $ npm install inquirer
      $ npm install fs
